

# منصة وظائف القطاع غير الربحي — خطة التنفيذ (MVP)

## نظرة عامة
منصة SaaS متخصصة بوظائف القطاع غير الربحي في السعودية والعالم العربي. تشمل موقعًا عامًا لعرض الوظائف، بوابة للجمعيات لإدارة إعلاناتها، ولوحة تحكم مركزية للإدارة والاعتماد.

---

## المرحلة 1: البنية التحتية وقاعدة البيانات

- إعداد Supabase (قاعدة بيانات + مصادقة + تخزين ملفات)
- إنشاء جداول: organizations, jobs, applications, plans, categories, user_roles, audit_logs, notifications
- إعداد نظام الأدوار (RBAC) مع جدول صلاحيات منفصل يدعم: Super Admin, Admin, Moderator, Org Owner, Org HR Manager, Org Viewer
- سياسات RLS لكل جدول حسب الدور
- إنشاء Storage buckets للشعارات وملفات CV

## المرحلة 2: نظام المصادقة والاشتراكات

- تسجيل دخول/خروج للجمعيات والإدارة
- نموذج طلب اشتراك جمعية (بيانات + مسؤول حساب)
- حالات الجمعية: قيد المراجعة → موافقة/رفض → تفعيل
- إدارة خطط الأسعار (عدد وظائف، Featured، Urgent، مقاعد، مدة الإعلان)
- ربط الجمعية بخطتها وحدود النشر

## المرحلة 3: الموقع العام (Public Website)

- **تصميم عربي RTL بالكامل، متجاوب Mobile-first، حديث واحترافي**
- الصفحة الرئيسية:
  - Hero مع بحث سريع + عداد الوظائف + أزرار CTA
  - قسم وظائف عاجلة (شريط علوي بارز)
  - قسم وظائف مميزة
  - تصنيفات الوظائف (بطاقات)
  - أحدث الوظائف
  - دليل الجمعيات (شعارات)
  - لماذا المنصة + كيف تعمل
  - باقات الأسعار مختصرة
  - Footer شامل
- صفحة الوظائف: بحث متقدم (كلمة، مدينة، تصنيف، نوع عمل، عن بُعد، خبرة، راتب، عاجلة/مميزة) + ترتيب + بطاقات احترافية مع Badges
- صفحة تفاصيل الوظيفة: وصف كامل + متطلبات + مسؤوليات + معلومات + CTA تقديم (نموذج داخلي أو رابط خارجي) + وظائف مشابهة + إبلاغ + مشاركة
- دليل الجمعيات + صفحة ملف الجمعية العام
- صفحات: الخدمات، الأسعار، اتصل بنا، الشروط والأحكام، سياسة الخصوصية

## المرحلة 4: بوابة الجمعيات (Organization Portal)

- Dashboard: وظائف نشطة/منتهية/مرفوضة + إحصاءات + تنبيهات
- إدارة الوظائف: إنشاء + مسودة + إرسال للاعتماد + تكرار + تمديد
- حالات الوظيفة: Draft → Submitted → Under Review → Approved/Rejected → Published → Expired/Archived/Suspended
- إدارة طلبات التقديم: عرض المتقدمين + تغيير الحالة (New/Reviewed/Shortlisted/Rejected/Hired) + تحميل CV + ملاحظات + تصدير CSV
- ملف الجمعية: الاسم، الشعار، المدينة، النبذة، الموقع، رقم الترخيص، روابط التواصل
- إدارة الفريق: إضافة مستخدمين + تعيين أدوار
- الاشتراك والفوترة: الخطة الحالية + تاريخ الانتهاء + ترقية

## المرحلة 5: لوحة التحكم المركزية (Admin Console)

- Dashboard: مؤشرات يومية/أسبوعية (وظائف، جمعيات، معدل موافقة، بلاغات)
- إدارة الوظائف: قائمة بفلاتر الحالة + شاشة مراجعة واعتماد مع Preview مطابق للموقع + Checklist + قرار + سبب رفض
- التحكم بـ Featured/Urgent/Boost
- إدارة التصنيفات: رئيسية/فرعية + ترتيب + تفعيل/تعطيل
- إدارة الجمعيات: حالات (Pending/Active/Suspended) + توثيق + مستخدمين تابعين
- طلبات الاشتراك: مراجعة + اعتماد + تفعيل الخطة
- إدارة الخطط والأسعار: بناء خطط + حدود + إضافات
- إدارة المستخدمين والصلاحيات + سجلات التدقيق (Audit Logs)
- البلاغات: بلاغ عن وظيفة/جمعية + إجراءات
- تحليلات أساسية: نمو الوظائف، أداء الجمعيات

## المرحلة 6: نظام التقديم والإشعارات

- نموذج تقديم داخلي (اسم، بريد، هاتف، CV، رسالة تغطية)
- تخزين ملفات CV بأمان (حد حجم + أنواع ملفات)
- إشعارات داخل النظام عند: إنشاء وظيفة، تغيير حالة، قرب إغلاق، وصول متقدم جديد

## المرحلة 7: الجودة والأمان

- قواعد النشر: لا تُنشر وظيفة بدون اعتماد، Featured/Urgent مرتبطة بالخطة
- Rate limiting + حماية النماذج
- سجلات تدقيق لكل موافقة/رفض/تعديل
- تصميم متجاوب ومحسّن للأداء

---

## ملاحظات تقنية
- Supabase Cloud للمصادقة وقاعدة البيانات والتخزين
- تصميم RTL عربي بالكامل باستخدام Tailwind CSS
- نظام أدوار في جدول منفصل مع Security Definer Functions
- الاشتراكات تبدأ يدوية (موافقة إدارية) مع إمكانية إضافة دفع إلكتروني لاحقًا
